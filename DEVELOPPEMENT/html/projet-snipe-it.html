Mise en place

J'ai tout d'abord mis en place Snipe-IT sur une machine virtuelle Linux.
La documentation officielle conseille d'utiliser les dernières versions d'Ubuntu, ou de Debian.
À terme, cette solution sera installé sur une machine virtuelle sur l'un des serveurs de mon entreprise. Ceci pour permettre un accès interne à celle-ci.

Extension

Lors de la mise en place de Snipe-IT, j'ai constaté un problème lors de l'ajouts de quantité à un accessoire.
Ceci n'étant pas très intuitif, j'ai décidé de coder une page web qui va permettre de faire cela automatiquement.

Il suffira donc de se rendre sur la page web contenant le formulaire, préciser la quantité, l'article, ainsi qu'un numéro de commande ; Ensuite, lors de la validation, le programme va s'occuper d'aller modifier ceci automatiquement sur Snipe-IT,
Il génère aussi un fichier texte dans lequel il va stocker toutes les données envoyées, pour avoir accès à des logs.
Pour réalisé ceci, j'ai utilisé Python pour l'automatisation ainsi que PHP pour l'exécution du script.
